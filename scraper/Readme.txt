To use the google image scraper the ChromeDriver (https://chromedriver.chromium.org/) and the Chrome binary (Chromium.exe or Chrome.exe) 
must be installed for your operating system. In the python file the path to these the Chrome Drivers and binary must be set. The Selenium
and urllib package must be installed.

The searchQuery var is the query that is used to browse google images. It is the same quary string as in a url quary string.
The downloadDir is the dir where the images will be downloaded to.
